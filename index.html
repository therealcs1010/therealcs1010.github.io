Recent Projects

Project 1

Automated Maze Navigation

We created a robot that could maneuver in a maze and overcome various obstacles using an array of sensors. Some of the tasks it could accomplish include colour detection using a Light-Dependent Resistor, audio frequency detection using a High-Pass Filter, as well as collision prevention using ultrasonic sensors.

Project 2

Remote Maze Navigation

We created a robot that could map its surrounding using a combination of a LiDAR sensor and the Hector SLAM algorithm. The robot was operated from a significant distance away using TCP, which provided communication between an on-board Raspberry Pi and the computer. The combination of sensors allowed us to remotely navigate the terrain without the use of a camera.

Project 3

AlgoSenpai's Adventures

This is a Java Desktop Application our team created which aims to bridge the learning gap that computing students face from their transition to university. Our application aims to teach fundamental data structures and algorithms to students who might not be proficient in programming. Our team decided to take a gamification approach to this problem to encourage new students to learn programming progressively.

Project 4

CNN Hardware Accelerator

For our Capstone Project, we created an automated dance teacher to encourage people to take up dancing during this Covid period. We use a combination of body sensors to detect the moves of the dancer and output a prediction on a laptop. For my part, I implemented a CNN (Convolutional Neural Network) Hardware Accelerator on an FPGA to speed up the predictions of the dance moves. This was done using a combination of Vivado HLS and Jupyter Notebook.

Project 5

Maplestory Private Server

This is an open-source version of Maplestory V176, which i was interested in maintaining and developing. Since this was an unfinished project, a lot of modifications were yet to be completed. The source code was done mainly in Java, while there were some scripts which required C++ and Python knowledge. This project has also challenged me as it forced me to pick up MySQL to learn how to maintain a database, as well as query items.

Project 6

FakeNews Telegram Bot

This is a Telegram Bot which our team implemented to help combat the rise of fake news on the Internet. Since people are unable to distinguish between real and fake news reliably, we experimented with several Machine Learning models such as LinearSVC, Naive Bayes, BERT Encodings, LSTMs to generate a reliable model that could predict if a news URL is real or fake given the article. This would then be displayed as a message on the front-end provided by Telegram's API.
