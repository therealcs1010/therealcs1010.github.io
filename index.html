Engineering Project

Automated Maze Navigation

​We created a robot which could manoeuvre in a maze and detect different obstacles such as different audio frequency using our constructed filter, different colours using an LDR, as well as anti-collision measures through ultrasonic and infrared sensors.

Engineering Project

LiDAR Remote Maze Navigation

​We created a robot which could map its surrounding using LiDAR technology. It could manoeuvre through a maze and use object detection to prevent a collision. This was done remotely using TCP, as which provided communication between an onboard Raspberry Pi and the computer. For the object detection, we channeled data from the LiDAR to the Arduino Board to the RPi before sending it to the laptop for us to run Hector Slam.
